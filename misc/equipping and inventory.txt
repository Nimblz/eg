STEPS TO EQUIP AN ITEM FOR DUMMIES (nimblz)

STEP1
    CHECK IF THE ASSET IS REAL

STEP2
    CHECK IF THE PLAYER OWNS THE ASSET

STEP3
    ADD ASSET TO FIRST IDX OF RETURN TABLE, LOOP THRU OLD EQUIPMENT 
    SLOTS 1..MAX-1 AND ADD TO RETURN TABLE AFTER FIRST IDX

HOW TO UNEQUIP AN ITEM

STEP1
    ITTERATE THRU EQUIPPED ITEMS
        IF ITS NOT THE ITEM TO REMOVE, INSERT INTO RETURN TABLE
        THIS HAS THE EFFECT OF REMOVING _ALL_ OF THE ITEM, BUT 
        THATS OKAY. MULTIPLE EQUIPS OF SAME ASSET SHOULD NOT BE 
        POSSIBLE
